# Personal Wealth & Expense Tracker 💰

A full-stack Personal Finance Command Center strictly designed to help users manage their income, track expenses, and deeply visualize their financial health through beautiful, dynamic UI charts.

### **🚀 Live Demo:** [https://wealth-expense.vercel.app](https://wealth-expense.vercel.app)

---

## 💻 Tech Stack
* **Frontend:** React, Vite, Vanilla CSS (Premium Glassmorphism UI)
* **Backend:** Java, Spring Boot, Spring Data JPA, REST APIs
* **Database:** TiDB Cloud (Distributed Serverless MySQL)
* **Deployment:** Vercel (Frontend) & Render (Backend)

## ✨ Technical Features
* **Real-time Synchronization:** Instantly add, calculate, or delete native transactions connected smoothly to a globally persistent TiDB cloud database.
* **Smart Visualization:** State-of-the-art dynamic Recharts auto-calculate to break down internal expense habits heavily by analytical categorization.
* **Premium Dashboard Design:** Built completely from scratch utilizing modern custom frosted-glass interfaces with sleek layout animations.
* **Robust Backend Infrastructure:** Strictly architected upon a scalable Controller-Service-Repository layer workflow using native Java 17 and Spring Boot protocols.
